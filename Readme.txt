***Welcome to AntiVPP (Antiviral-Peptide-Predictor). AntiVPP is a roboust program to predict antiviral peptides with a friendly interface.

Steps to run AntiVPP:

1- Download all the AntiVPP files

2- Install WinPython 3.6.6.2Qt5-64 bit (*) available at: https://winpython.github.io/

3- Copy and paste the AntiVPP files into the installed WinPython software

4- Locate and run the AntiVPP.py file
